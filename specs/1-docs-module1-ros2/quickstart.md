# Quickstart: Docusaurus Documentation Site

This guide explains how to set up and run the Docusaurus documentation site locally.

## Prerequisites

- Node.js (v20 or later)
- npm or yarn

## Installation

1.  **Navigate to the docs directory**:
    ```bash
    cd docs
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```
    or
    ```bash
    yarn install
    ```

## Running the Site

1.  **Start the development server**:
    ```bash
    npm start
    ```
    or
    ```bash
    yarn start
    ```

2.  **Open in browser**:
    The site will be available at http://localhost:3000.
