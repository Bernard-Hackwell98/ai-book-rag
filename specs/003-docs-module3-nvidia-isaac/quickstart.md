# Quickstart: Module 3 – The AI-Robot Brain (NVIDIA Isaac™)

This quickstart guide provides instructions on how to access the documentation for Module 3.

## Prerequisites

-   A web browser.
-   An internet connection.

## Accessing the Documentation

1.  **Navigate to the documentation website.** (The URL will be provided once the documentation is deployed).
2.  **Navigate to Module 3.** In the sidebar, you will find a new section for "Module 3 – The AI-Robot Brain (NVIDIA Isaac™)".
3.  **Explore the chapters.** The module is divided into three chapters:
    -   Chapter 1: NVIDIA Isaac Sim
    -   Chapter 2: Isaac ROS
    -   Chapter 3: Navigation and Planning

Enjoy learning about the AI-Robot Brain!
